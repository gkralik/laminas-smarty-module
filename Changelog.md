# Changelog

## [4.0.0] - 2025-02-11

### Added

- Adopt PER-CS2.0 and Symfony coding style

## [3.0.0] - 2024-03-07
### Added
- Initial version (based on [gkralik/zf3-smarty-module](https://github.com/gkralik/zf3-smarty-module)).
